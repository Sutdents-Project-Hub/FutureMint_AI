## 建議 Demo 主線

1. Persona 收到零用錢，卻沒有先分配目標。
2. 輸入「今天午餐跟飲料 185」，AI 拆成結構化草稿並顯示信心與待確認欄位。
3. 使用者確認後，確定性規則更新本週預算與目標差距。
4. 系統以已記錄的訂閱價格與分帳人數，比較合成方案的每月負擔、節省差額與資格。
5. AI 把這次事件轉成一堂短課，說明機會成本與下一步行動。
6. FutureSeed 用公式顯示「把每月一部分結餘留下」的情境，清楚標示教育模擬。

## Azure PoC 最小證據

- 已建立 30 筆全合成繁中 deterministic-provider 測試集，包含收入、單筆／多項支出、相對日期、缺少金額、訂閱、分帳、否定句與無關輸入；目前 30/30 完整通過。此結果不冒充 Azure OpenAI 成效。
- 記錄 schema 合法率、欄位正確率、平均／P95 延遲、429 或 timeout 次數與降級結果。
- 比較 AI 前後：原句、結構化結果、人工修正與最後保存資料。
- 展示 Functions 日誌與 Application Insights 指標時，只顯示去識別摘要，不暴露 prompt、金鑰或個人財務全文。
- 所有成效數字都標記「實測」或「目標」；不把推估包裝成既有成果。

- 正常路徑：Web／Android 連線至 Azure 測試環境。
